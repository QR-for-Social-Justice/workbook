For example, the data in total co2 emissions of the USA and China [Excel Spreadsheet] 72  is from the World Bank. The data gives the CO2 emissions of the United States and China from 1960 to 2018 in kilotons. Because these data sets have a similar scale, we can display them on a single graph. We use a line graph because the data set is quantitative continuous and has a single value for each year.
